About AMGR SSO in STP
-------------------------
This module allows external vendors to do SSO in STP site using the accessToken received through AMGR oauth API.


I. FEATURES:

1. Allow account manager users to login into STP using the access token

2. Deep link supported for the STP

3. Reverse SSO supported for classic AMGR

4. Reverse SSO supported for AMGR

See INSTALL.txt for installation and administration instructions.
